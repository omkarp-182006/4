Selenium Setup and Selenium IDE Extension Guide
(Python)
1. Install Python
1 Go to the official Python website: https://www.python.org
2 Download the latest version of Python.
3 Run the installer.
4 Enable the option 'Add Python to PATH'.
5 Click 'Install Now' to complete installation.
Verification Command:
python --version


2. Install Selenium Library
1 Open Command Prompt or Terminal.
2 Install Selenium using pip.
pip install selenium
Verify Installation:
pip show selenium

3. Install ChromeDriver (WebDriver)
1 Open Google Chrome.
2 Check Chrome version using: chrome://settings/help
3 Download the matching ChromeDriver from https://chromedriver.chromium.org/downloads
4 Extract the downloaded file.
5 Place chromedriver.exe inside your project folder or Python Scripts folder.

4. Install Selenium IDE Extension
1 Open Google Chrome browser.
2 Go to Chrome Web Store: https://chrome.google.com/webstore
3 Search for 'Selenium IDE'.
4 Click 'Add to Chrome'.
5 Confirm by clicking 'Add Extension'.
6 After installation, the Selenium IDE icon will appear in the browser toolbar.

5. How to Use Selenium IDE
1 Click the Selenium IDE icon in Chrome.
2 Click 'Create New Project'.
3 Enter a project name.
4 Provide the base URL of the website.
5 Click 'Start Recording'.
6 Perform actions on the website (click, type, navigate).
7 Stop recording and save the test case.

6. Basic Python Script to Verify Selenium
from selenium import webdriver
driver = webdriver.Chrome()
driver.get("https://www.google.com")
print(driver.title)
driver.quit()
